### Про проект
Цей проєкт є веб-додатком, який реалізує резюме автора. 
Вона складається з:
* Особистої інформації автора
* Навички автора
* Блок опиту роботи
* Проекти автора
* Контакти для зв'язку

Це резюме відрізняється від інших тим, що має унікальний дизайн, який є анімованим
